# 🚀 Sobre o Projeto

Este projeto foi desenvolvido com o objetivo de praticar e evoluir habilidades em desenvolvimento front-end, criando um portfólio com aparência profissional e moderna.

O site possui:
- Design responsivo 📱
- Layout moderno e minimalista 🎨
- Efeitos suaves e animações em CSS ✨
- Fundo grid estilizado 🌌
- Navegação suave entre sections ⚡
- Estrutura organizada 📂
- Ícones Font Awesome 🔥
